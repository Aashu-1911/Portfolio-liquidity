# Portfolio-liquidity
This project is a machine-learning based portfolio liquidity prediction platform that estimates current liquidity risk, forecasts near-term liquidity changes, and explains those forecasts to support smarter trading and risk decisions.

## Features
- Portfolio liquidity scoring
- ML-based liquidity prediction
- Future liquidity forecasting
- Support for US (S&P 500) and Indian (NIFTY) markets

## Tech Stack
Frontend: React + Vite + Tailwind  
Backend: Python + Flask  
ML: Scikit-learn  
Data: S&P500 + NIFTY historical data

## How to Run
1. pip install -r requirements.txt
2. python app.py
3. npm install
4. npm run dev
